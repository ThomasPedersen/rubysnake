
# Ruby Snake!

The classic snake-game running on Ruby using Gosu.

## Features

 - Fantastic growing snake
 - Apples to eat to your heart's content

## Installation

	$ gem install gosu
	$ git clone https://github.com/jessc/rubysnake.git

## Play

	$ ruby rubysnake.rb

## Commands

 - Move: Arrow Keys
 - Pause: Space
 - Quit: Escape or Command+Q
